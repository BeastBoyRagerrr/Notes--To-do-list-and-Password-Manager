# src
A Java-based Personal Manager application with a graphical user interface for managing notes, tasks, and passwords in one place.  Features Notes Management: Create, edit, and organize personal notes To-Do List Management: Add, update, and track tasks Password Manager: Store and manage passwords securely Java Swing GUI: Interactive desktop interface Modular Architecture: Separate managers for each functionality File Handling: Persistent storage using file I/O Exception Handling: Robust validation and error management OOP-Based Design: Clean and reusable object-oriented structure
